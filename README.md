## Module 5: Correlation and Regression Analysis 
### of Healthcare Charges

### Overview
Analysis of healthcare insurance charges using 
a dataset of 1,338 individuals examining 
relationships between demographic variables, 
lifestyle factors, and medical costs.

### Methods Used
- Covariance Analysis
- Pearson Correlation
- Hypothesis Testing
- OLS Regression Modeling
- Density Plot Visualization
- Residual Diagnostics

### Key Findings
- Annual Salary strongest predictor (r = 0.95)
- Daily Steps highly significant (r = 0.89)
- OLS Model R-squared = 0.992
- Smoking status creates clear cost divide
- 6 of 7 predictors statistically significant

- Python (pandas, numpy, matplotlib, seaborn, statsmodels)
